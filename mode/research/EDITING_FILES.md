
EDITING FILES

# 适用范围
**仅文本文件**

# 编辑方式
通过调用 [ read_file / write_to_file / replace_in_file ] 工具调用，你可以 [ 读 / (新建/重写/追加) / 按块修改 ] **文本文件**

# 编辑策略
如果文件较大或需要修改的内容较多（如超过50行），应尽量分多次构建。如当需要创建一个100行的文件时，可以先以new模式调用 write_to_file 创建文件，并先写入20行内容，然后再通过append模式调用 write_to_file ，分4次每次追加20行内容。注意完成后，应调用 read_file 验证生成文件的内容。
