## 三、供需基本面分析

### 3.1 供应端分析

#### （一）国内产量与种植面积

**2025年国产玉米情况（影响2026年上年度供应）：**
- 2025年全国玉米播种面积约 **6.8亿亩**，同比基本持平
- 2025年玉米产量约 **2.89亿吨**，较上年小幅增加
- 主要产区（黑龙江、吉林、山东、河南）气候条件总体正常

**2026年新季玉米种植预期：**
- 春播即将展开，大豆玉米种植面积调整政策仍具不确定性
- 种植成本方面：地租、化肥、种子等成本居高不下，支撑远期价格底部
- 政策面："稳大豆、扩油菜"政策导向可能制约玉米面积大幅增长

#### （二）进口情况

- 2025年度玉米进口量约 **1,200万吨**，较峰值有所回落
- 2026年配额内进口仍为720万吨，配额外关税较高
- 进口来源仍以巴西、美国为主，但国际玉米价格波动较大
- 人民币汇率波动对进口成本产生影响

#### （三）库存水平

- 临储库存消化完毕，当前为市场化库存
- 渠道库存（贸易商+深加工企业）处于**中等偏低水平**
- 2026年初以来价格上涨背景下，渠道补库意愿增强，进一步推升价格

### 3.2 需求端分析

#### （一）饲用消费（占比约65%）

玉米最大需求端为饲料消费：

- **生猪存栏**：2026年生猪存栏仍处高位，能繁母猪存栏充足，猪料需求持稳
- **禽料需求**：白羽肉鸡、蛋鸡存栏量均处高位，禽料需求旺盛
- **水产料**：4月后水产养殖逐步启动，季节性需求回升
- 综合判断：2026年饲料消费总量预计 **稳中有增**，增幅约1-2%

#### （二）深加工消费（占比约30%）

- **淀粉加工**：淀粉加工利润改善，开工率回升至60-70%
- **酒精/乙醇**：燃料乙醇需求稳定，但受制于粮食乙醇政策限制
- **氨基酸/赖氨酸**：出口需求向好，支撑深加工产能利用率
- 深加工总消费量预计稳中略增

#### （三）食用消费及其他（占比约5%）

基本稳定，占比小，影响有限。

## 四、价格驱动因素分析

### 4.1 利多因素（支撑价格上涨）

| 因素 | 影响程度 | 具体分析 |
|:----|:-------:|:---------|
| **渠道库存偏低** | ⭐⭐⭐⭐⭐ | 临储去库存完成，社会库存处于偏低水平，贸易商补库意愿强 |
| **饲用需求韧性** | ⭐⭐⭐⭐ | 生猪和禽类存栏高位，饲料消费刚性增长 |
| **进口下降** | ⭐⭐⭐ | 国际价格高企及配额限制，进口量同比下降 |
| **种植成本支撑** | ⭐⭐⭐ | 地租、农资、人工成本居高不下，底部支撑明确 |
| **深加工需求增长** | ⭐⭐⭐ | 淀粉、氨基酸等行业产能持续扩张 |
| **国际市场联动** | ⭐⭐ | 全球玉米供需偏紧，CBOT玉米维持高位 |

### 4.2 利空因素（压制价格上涨）

| 因素 | 影响程度 | 具体分析 |
|:----|:-------:|:---------|
| **新作增产预期** | ⭐⭐⭐⭐ | 2026年新季玉米播种面积稳中有增，下半年供应将改善 |
| **政策调控** | ⭐⭐⭐ | 国家通过定向拍卖、进口调节等手段抑制价格过快上涨 |
| **替代品竞争** | ⭐⭐⭐ | 小麦、稻谷等替代品价格优势抑制玉米饲用需求上限 |
| **宏观需求疲软** | ⭐⭐ | 经济复苏节奏偏慢，消费恢复不及预期 |
| **猪周期下行风险** | ⭐⭐ | 若猪价持续低迷，养殖端去产能将抑制远期饲用需求 |

### 4.3 近期关键事件（2026年4月）

1. **4月21-22日放量拉升**：C2607两日累计上涨约30元/吨，成交量从70万手放大至94万手，突破2,400关口
2. **港口库存变化**：北港（锦州港、鲅鱼圈）玉米库存小幅下降，南港（广东港）库存中等偏下
3. **深加工企业提价收购**：华北深加工企业连续上调挂牌价，提振市场信心
4. **市场情绪**：贸易商看涨预期浓厚，出货节奏偏慢，渠道囤货意愿增强

- 新季玉米上市前为 **传统青黄不接期**，供应偏紧格局难以根本改变
- C2607合约作为主力合约，2,350-2,450为主要波动区间

- 新作上市后供应改善，远月合约（C2611、C2701）已提前反映贴水
- 新作产量是关键变量，需持续跟踪面积和天气
- 若产量正常，价格中枢预计下移至**2,200-2,350元/吨**

## 七、风险提示

1. **政策风险**：国家可能通过增加进口配额、定向销售储备等方式平抑价格
2. **天气风险**：东北春旱、夏涝等极端天气对新作产量的影响具有重大不确定性
3. **国际市场风险**：CBOT玉米价格、国际贸易摩擦、汇率变动等因素影响进口成本
4. **替代品风险**：小麦、稻谷等替代品与玉米比价关系的变化可能改变饲用配方
5. **疫情/疫病风险**：非洲猪瘟等疫病暴发可能导致饲用需求骤降
6. **数据局限性**：本报告供需平衡表数据为估算值，可能与官方最终数据存在差异
