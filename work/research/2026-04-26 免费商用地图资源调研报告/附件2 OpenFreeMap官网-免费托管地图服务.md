# 附件2 OpenFreeMap官网-免费托管地图服务

## 原始来源

- **网站：** OpenFreeMap 官方网站（https://openfreemap.org/）
- **采集日期：** 2026年4月26日

---

## 一、平台概况

OpenFreeMap 是一个 **完全免费、开源** 的地图瓦片托管服务，基于 OpenStreetMap 数据。

| 项目 | 说明 |
|------|------|
| 创始人 | 个人项目（创始人拥有9年地图基础设施经验） |
| 数据来源 | OpenStreetMap |
| 服务模式 | **免费公共实例** + **自托管** |
| 许可协议 | **完全开源**（无"开放核心"模式） |

## 二、免费政策

### 公共实例完全免费

> "There are no limits on the number of map views or requests. There's no registration, no user database, no API keys, and no cookies."

| 限制项 | 是否有上限 |
|:------|:--------:|
| 地图加载次数 | ❌ **无限制** |
| 请求次数 | ❌ **无限制** |
| 用户注册 | ❌ 无需注册 |
| API Key | ❌ 不需要 |
| Cookie | ❌ 不使用 |

### 自托管方案

提供完整的开源部署方案，包括：
- 完整的地球数据下载（Btrfs 和 MBTiles 格式）
- 默认样式表
- Quick Start Guide

## 三、数据提供方式

| 方式 | URL/说明 |
|:----|:---------|
| **瓦片API** | 基于 OSM 数据渲染的底图瓦片 |
| **样式** | 开源样式：https://github.com/hyperknot/openfreemap-styles |
| **自托管** | 完整生产环境部署脚本 |
