## 四、MapLibre（开源引擎）

| 项目 | 说明 |
|:----|:-----|
| 网站 | https://maplibre.org/ |
| 特点 Mapbox GL JS 的开源分支 |
| 许可 | **BSD 3-Clause**（完全开源免费）|
| 使用方式 | 自托管，可搭配任何瓦片源 |
| 免费商用 | ✅ 完全允许 |

**核心价值：** MapLibre 本身不是地图数据源，而是一个**开源地图渲染引擎**，可搭配 OSM/天地图/自有数据 使用。结合 Protomaps 或自托管的 OSM 瓦片，可实现完全免费的自主地图方案。

## 六、免费方案推荐组合

| 需求场景 | 推荐组合 | 成本 |
|:--------|:--------|:---:|
| 中国底图+地址 | **天地图**（底图+地址API） | **免费** |
| 全球底图 | **Protomaps + 自托管S3** | ~$10-20/月 |
| 全球底图+地址 | **OSM Nominatim（自托管）+ Stadia Maps** | ~$0-49/月 |
| 欧美精确地址 | **Stadia Maps 免费套餐** | **免费** |
| 企业级全球方案 | **Mapbox（付费）+ 天地图（中国区）** | 按量付费 |
| 全开源自建 | **OSM + MapLibre + Protomaps** | 服务器成本 |
| 中国地址查询 | **天地图 API** | **免费** |
| 全球底图+中国地址 | **Protomaps（底图）+ 天地图（地址）** | **最小成本** |
