##一、平台概况

OpenStreetMap（OSM）是一个由全球志愿者共同创建和维护的**自由开源的地图数据库**，成立于2004年。截至2026年，OSM社区拥有超过**1,000万注册用户**，每日产生约**300万次编辑**。

**核心网址：**
- 主站：https://www.openstreetmap.org
- 数据下载：https://planet.openstreetmap.org
- 维基：https://wiki.openstreetmap.org
- API：https://api.openstreetmap.org

## 二、许可协议 — ODbL

OpenStreetMap 的数据基于 **Open Data Commons Open Database License (ODbL)** 许可协议发布。

### ODbL 核心要点

| 条款 | 说明 |
|------|------|
| ✅ **允许自由使用** | 可复制、分发、传输和改编数据 |
| ✅ **允许商用** | **明确允许商业用途** |
| ✅ **允许再分发** | 可整合到其他产品或服务中 |
| ⚠️ **要求署名（Attribution）** | 必须注明 "© OpenStreetMap contributors" |
| ⚠️ **需以相同方式共享（Share-Alike）** | 若对数据进行改编并公开分发，需以ODbL许可发布 |
| ⚠️ **保持开放（Keep Open）** | 不可通过技术手段限制他人使用数据 |

> **出处：** OpenStreetMap 版权页面 https://www.openstreetmap.org/copyright

## 三、数据覆盖与质量

| 维度 | 说明 |
|:----|:------|
| **全球覆盖** | 全球220+个国家和地区都有数据 |
| **道路网络** | 全球道路总里程超过 **6,000万公里** |
| **地址数据** | 欧美发达国家地址非常详尽；中国等地区详略不一 |
| **POI数据** | 全球POI超过 **1亿个** |
| **更新频率** | 实时更新（社区持续编辑） |
| **底图质量** | 欧美：与Google Maps相当甚至更详细；中国：不如高德/百度/天地图 |
| **地址查询精度** | 欧美较高（可达门牌号级）；中国地址精确度一般 |

### 中国地区数据质量评价

| 维度 | 评分（满分5） | 评价 |
|:----|:----------:|:-----|
| 道路网络 | ⭐⭐⭐ | 主要道路完整，次级道路有缺失 |
| 地址编码 | ⭐⭐ | 大城市较好，中小城市地址精度不足 |
| POI丰富度 | ⭐⭐⭐ | 一二线城市POI较多，三四线较少 |
| 更新时效 | ⭐⭐⭐ | 大城市更新较快，偏远地区滞后 |
| 底图数据 | ⭐⭐⭐ | 可作为底图使用，但细节不如国产地图 |

## 四、数据获取方式

| 方式 | 说明 | 适用场景 |
|:----|:-----|:--------|
| **Planet.osm 完整下载** | 完整数据集（约150GB压缩） | 离线全量分析 |
| **Overpass API** | 按区域/条件查询 | 按需获取特定区域数据 |
| **Tile 服务器** | 渲染瓦片图 | 网页/APP地图显示 |
| **OSM Nominatim API** | 地理编码（地址→坐标） | 地址查询 |
| **OSM 数据提取** | geofabrik.de 提供区域提取 | 按国家/地区下载 |

## 五、资质评价

| 维度 | 评价 |
|:----|:------|
| **运营资质** | OSM 基金会（OpenStreetMap Foundation）运营，英国注册非营利组织 |
| **数据权威性** | 众包模式，非官方测绘数据，权威性低于政府测绘数据 |
| **数据可靠性** | 社区审核 + 自动化质量检查机制，成熟度高 |
| **资质证书** | 无商业测绘资质 |
| **适合场景** | 全球底图、欧美地址查询、户外导航、学术研究 |

## 六、结论

- ✅ **免费商用**：明确允许商业用途（ODbL许可）
- ✅ **数据可靠**：20+年积累，全球最大众包地图项目
- ✅ **技术成熟**：丰富的API和工具生态系统
- ⚠️ **中国数据有限**：中国区地址精度不如国产地图
- ⚠️ **需要署名**：必须标注数据来源
- ⚠️ **Share-Alike**：若改编数据需以同样许可发布
