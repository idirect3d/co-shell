## 原始来源

- **网站：** Protomaps 官方网站（https://protomaps.com/）

## 一、平台概况

Protomaps 提供一种**基于单个文件（PMTiles）** 的开源地图方案，将整个地球的地图数据打包成单个静态文件，可直接部署在云存储上。

| 项目 | 说明 |
|:----|:-----|
| 数据来源 | OpenStreetMap |
| 核心产品 | PMTiles 格式 + 命令行工具 |
| 许可协议 | **开源**（BSD 3-Clause） |
| 方式 | 自托管（S3/Cloudflare/任何云存储） |

## 二、核心技术 — PMTiles

PMTiles 是一种 **Hilbert 排序的瓦片金字塔归档** 格式，支持：

| 特性 | 说明 |
|----|:-----|
| 单文件 | 整个地球矢量瓦片在**一个文件**中 |
| HTTP Range | 利用 HTTP 范围请求按需加载 |
 云原生 | 直接部署在 S3/Cloudflare R2 等 |
| 无需服务器 | 不需要专门的瓦片服务器 |
| 支持叠加层 | 自定义地理数据可叠加 |
