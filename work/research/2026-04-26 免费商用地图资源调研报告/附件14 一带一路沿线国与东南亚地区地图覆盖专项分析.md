## 二、各平台在东南亚的覆盖与精度

东南亚是"一带一路"核心区域，也是中国企业出海最密集的地区。

### 2.1 OSM 在东南亚的数据质量

| 国家 | OSM数据质量 | 道路覆盖率 | POI丰富度 | 活跃社区 |
|:----|:----------:|:---------:|:---------:|:-------:|
| **新加坡** | ⭐⭐⭐⭐⭐ | 95%+ | 丰富 | 非常活跃 |
| **马来西亚** | ⭐⭐⭐⭐ | 80-90% | 较丰富 | 活跃 |
| **泰国** | ⭐⭐⭐⭐ | 80-85% | 较丰富 | 活跃 |
| **越南** | ⭐⭐⭐⭐ | 75-85% | 中等 | 活跃 |
| **印尼** | ⭐⭐⭐ | 60-75% | 中等 | 较活跃 |
| **菲律宾** | ⭐⭐⭐ | 60-70% | 中等 | 较活跃 |
| **柬埔寨** | ⭐⭐⭐ | 50-65% | 较少 | 发展中 |
| **老挝** | ⭐⭐ | 40-55% | 较少 | 发展中 |
| **缅甸** | ⭐⭐ | 30-50% | 较少 | 受限 |

> **总结：** 新加坡和马来西亚的OSM数据质量很高，可媲美欧美发达国家。越南、泰国也能满足基本需求。柬埔寨、老挝、缅甸数据较薄弱。

### 2.2 天地图的东南亚覆盖

天地图以中国数据为核心，东南亚地区的覆盖情况：

| 维度 | 评价 |
|:----|:------|
| **底图数据** | ⭐⭐ 基于合作数据，准确性一般 |
| **道路数据** | ⭐⭐⭐ 主要城市主要道路可用 |
| **地址编码** | ⭐⭐ 无法精确到门牌号 |
| **POI** | ⭐⭐ 数量有限，不如OSM |

> **结论：** 天地图的优势在中国境内，东南亚地区不推荐作为主力。

### 2.3 Stadia Maps / MapTiler 等OSM基础平台

基于OSM数据，因此东南亚覆盖度与OSM一致：
- 新加坡/马来西亚：⚠⭐⭐⭐⭐
- 越南/泰国：⭐⭐⭐⭐
- 印尼/柬埔寨/老挝/缅甸：⭐⭐⭐

### 2.4 Mapbox 在东南亚的数据

Mapbox 在OSM基础上进行了大量增强，数据质量高于纯OSM：

| 国家 | Mapbox 质量 | 与OSM对比 |
|:----|:----------:|:---------|
| 新加坡 | ⭐⭐⭐⭐⭐ | 显著增强 |
| 马来西亚 | ⭐⭐⭐⭐⭐ | 明显增强 |
| 泰国 | ⭐⭐⭐⭐ | 有所增强 |
| 越南 | ⭐⭐⭐⭐ | 有所增强 |
| 印尼 | ⭐⭐⭐⭐ | 有所增强 |
| 柬埔寨/老挝/缅甸 | ⭐⭐⭐ | 部分增强 |

## 六、推荐方案（补充一带一路/东南亚需求）

### 场景一：东南亚地区业务

| 方案 | 组件 | 月成本 | 说明 |
|:----|:-----|:------|:-----|
| **推荐方案** | **Stadia Maps**（底图+地址） | **免费**（50万次内） | 新加坡/马来西亚/泰国覆盖好 |
| 增强方案 | **Mapbox**（底图+地址） | 5万次免费，超出付费 | 东南亚数据有增强 |
| 自建方案 | **Protomaps**（底图）+ **Nominatim自托管**（地址） | ~$10-20/月 | 适合大规模应用 |

### 场景二：一带一路全区域覆盖

| 优先顺序 | 平台 | 理由 |
|:-------:|:----|:------|
| 🥇 | **Stadia Maps / OSM基础** | 免费额度大，全球覆盖一致性好 |
| 🥈 | **Mapbox** | 各地都有数据增强，质量最高 |
| 🥉 | **Protomaps自托管** | 成本可控，不依赖外部API |

### 场景三：中国企业出海（东南亚+一带一路）

| 组件 | 推荐平台 | 月成本 | 覆盖区域 |
|:----|:--------|:------|:--------|
| **全球/东南亚底图** | Stadia Maps（免费套餐） | **免费** | 东南亚⭐⭐⭐⭐ |
| **中国底图** | 天地图 | **免费** | 中国⭐⭐⭐⭐⭐ |
| **东南亚地址查询** | Stadia Maps Geocoding | **免费**（25万次/月） | 东南亚⭐⭐⭐ |
| **中国地址查询** | 天地图 | **免费** | 中国⭐⭐⭐⭐ |

### 最佳组合推荐

```
中国企业出海项目地图方案：
┌─────────────────────────────────────────────────┐
│ 中国区     → 天地图（底图 + 地址查询）            │ 免费
│ 东南亚区   → Stadia Maps（底图 + 地址查询）       │ 免费（额度内）
│ 其他地区   → Stadia Maps 或 Mapbox              │ 免费/按需
│ 统一渲染   → MapLibre GL JS（开源渲染引擎）       │ 免费
└─────────────────────────────────────────────────┘
此方案：中国地址查询最准 + 东南亚免费覆盖 + 全球统一体验
```

## 七、关键结论

1. **东南亚地图覆盖最佳平台**：**Stadia Maps/OSM基础平台**（免费方案中首选）
2. **新加坡、马来西亚**的OSM数据质量**接近欧美水平**
3. **越南、泰国、印尼**的大中城市覆盖尚可，但门牌号地址精度有限
4. **柬埔寨、老挝、缅甸**数据较薄弱，需使用Mapbox或其他增强数据
5. **一带一路中东/非洲国家**：OSM人道主义团队有大量贡献，基本覆盖可用
6. **天地图仅在中国境内有优势**，一带一路和东南亚地区不推荐作为主力
7. **中国企业出海的最佳实践：天地图（中国）+ Stadia Maps/OSM（海外）**
