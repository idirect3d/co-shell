# 关于蜂蜜松露甜蛋白（Honey Truffle Sweet Protein）甜味剂市场的调研报告


**调研发〔2026〕1号**

---

各有关单位：

蜂蜜松露甜蛋白（Honey Truffle Sweet Protein，以下简称HTSP）是一种源自匈牙利蜂蜜松露（*Mattirolomyces terfezioides*）的新型天然甜味蛋白，由美国MycoTechnology公司采用精密发酵技术规模化生产，商业化品牌名为"Zukora"。该产品比蔗糖甜1000至2500倍，热量极低，于2026年3月获得美国自确认GRAS（一般公认安全）身份，正式进入美国市场。

为深入了解该产品的技术路线、市场竞争格局、发展趋势及潜在影响，调研工作组通过文献检索、网络调研、数据分析，对HTSP甜味剂市场进行了系统调研，现将有关情况报告如下。

---

## 一、产品基本概况

### （一）产品定义与特性

蜂蜜松露甜蛋白是一种从稀有蘑菇——蜂蜜松露中分离出的天然甜味蛋白质。其核心属性如下：

1. **来源**：蜂蜜松露（*Mattirolomyces terfezioides*），原产于匈牙利等地，已有悠久的食用历史【附件7】。
2. **有效成分**：HT-AC（Honey Truffle Active Compound），由121个氨基酸组成的甜味蛋白质，喷雾干燥粉末中总蛋白含量约50%【附件3】。
3. **甜度**：根据应用场景不同，比蔗糖甜1000至2500倍【附件2】【附件4】。
4. **热量**：作为非营养性高强度甜味剂，几乎不贡献热量【附件1】。
5. **口感**：具有类似蔗糖的纯净甜味，无部分天然甜味剂（如甜菊糖）常见的苦涩余味【附件4】。
6. **安全性**：经细菌回复突变试验、哺乳动物细胞微核试验及13周口服重复剂量毒性研究（大鼠）验证，无毒、不致敏、可被人体胃肠道完全消化。未观察到有害作用的剂量（NOAEL）为5400 mg/kg/day（相当于HT-AC 1500 mg/kg/day）【附件3】。

## 二、技术路线与生产工艺

### （一）核心技术平台

MycoTechnology公司采用**精密发酵（Precision Fermentation）**技术生产HTSP，该技术路线分为以下关键步骤【附件5】：

1. **发现与基因解析**：从天然蜂蜜松露中识别并分离出负责甜味的蛋白质，确定其完整DNA序列（编码121个氨基酸）。
2. **宿主构建**：将甜味蛋白编码基因插入*Komagataella phaffii*（亦称毕赤酵母*Pichiais*）表达系统中【附件3】。
3 **深层发酵**：通过优化培养基配方、温度、pH等工艺参数，在3000升发酵罐中进行高密度发酵【附件2】【附件7】。
4. **下游纯化**：采用先进的分离纯化技术提取目标蛋白。
5. **喷雾干燥**：将纯化后的蛋白溶液制成粉末状成品。

### （二）技术优势

1. **可持续性**：与传统蔗糖生产相比，水资源和土地利用率显著更高【附件6】。
2. **清洁标签**：天然来源的非化学合成产品，符合清洁标签趋势【附件7】。
3. **可放大性**：已成功实现从实验室到商业批次的工艺放大【附件5】。
4. **平台化能力**：公司同时开发了ClearIQ™（风味调节剂）、ClearHT（风味增强剂）和FermentIQ™（菌丝体发酵植物蛋白）等多条产品线，FAAS（发酵即服务）平台具有跨产品延展性【附件2】。

### （三）蜂蜜松露甜蛋白的竞争优劣势

**竞争优势：**

1. **品类首创性**：被业界认为是"几十年来发现的第一种新型天然甜味剂"【附件7】。
2. **口感纯净度**：与甜菊糖等竞品相比，无不良余味【附件4】。
3. **安全性数据充分**：拥有在同行评审期刊（*Food and Chemical Toxicology*）发表的安全性评价数据【附件3】。
4. **GLP-1时代契合度**：不影响血糖水平，适合配合GLP-1类减肥药物的饮食方案【附件5】。
5. **独特的故事性**：源自稀有蜂蜜松露，具有天然的消费者吸引力。

**竞争劣势：**

1. **生产成本较高**：精密发酵工艺的初期成本高于传统提取或化学合成。
2. **市场认知度低**：作为全新品类，消费者教育和市场推广需大量投入。
3. **监管审批待完善**：除美国自确认GRAS外，欧盟、中国、日本等主要市场的审批尚在推进中。
4. **热稳定性局限**：蛋白质类甜味剂在高温和酸性条件下的稳定性需要持续优化。
5. **产能尚未充足**：当前生产规模有限，尚不能满足大规模市场供应需求。

## 四、市场规模与增长预测

### （一）全球甜味蛋白市场总体规模

根据国际市场研究机构Persistence Market Research于2026年3月发布的报告【附件8】：

| 指标 | 数据 |
|------|------|
| 2026年市场规模 | 1340万美元 |
| 2033年预计市场规模 | 2660万美元 |
| 复合年增长率（CAGR） | 10.3% |
| 预测期间 | 2026年—2033年 |

### （二）市场增长驱动因素

1. **健康意识持续提升**：世界卫生组织（WHO）建议将游离糖摄入量降至每日总热量的10%以下，理想目标为5%以下，推动食品企业寻求替代方案【附件3】。
2. **监管政策趋严**：全球多国实施糖税政策，促进低糖/无糖产品的开发与推广【附件8】。
3. **清洁标签运动深入**：消费者日益倾向于选择天然、配料简单的食品，天然甜味蛋白符合这一趋势【附件7】。
4. **GLP-1药物市场爆发**：GLP-1受体激动剂类减肥药的广泛应用，催生了"减肥友好型"食品配方的巨大需求，甜味蛋白因不影响血糖、有益肠道健康而受益显著【附件5】。
5. **技术成熟推动成本下降**：精密发酵技术的持续进步和规模化生产，正在降低甜味蛋白的生产成本。

### （三）蜂蜜松露甜蛋白市场前景预测

基于行业数据合理推断：

1. **初期阶段（2026—2028年）**：Zukora刚进入市场，主要面向高端食品饮料品牌，预计在甜味蛋白市场中占据5%—10%的份额。
2. **成长期（2028—2030年）**：产能扩大、成本下降、多国监管获批后，预计市场份额提升至15%—20%，年增长率可达15%—25%。
3. **成熟期（2030年后）**：若技术持续突破且市场接受度良好，有望成为甜味蛋白市场的主要参与者之一。

## 五、监管审批现状

 （一）美国市场

MycoTechnology已于2026年月获得**自确认GRAS**（Self-affirmed Generally Recognized as Safe）身份，产品Zukora可合法进入美国市场销售。同时，公司已向FDA提交正式GRAS通知，预计FDA出具"无问题"函件需9至18个月【附件5】。

### （二）欧盟市场

MycoTechnology旗下菌丝体产品FermentIQ™系列此前已获得欧盟"新型食品"（Novel Food）授权。蜂蜜松露甜蛋白在欧盟的审批需单独提交新型食品申请，目前尚未有公开信息显示具体进展。

### （三）中国市场

蜂蜜松露甜蛋白作为新型食品原料，需按照《食品安全法》及相关法规向国家卫生健康委员会提交安全性评价资料。鉴于中国对新型食品原料的审慎态度，预计审批周期较长。

### （四）其他市场

日本、韩国、澳大利亚、巴西等主要食品市场同样需要分别提交相应的食品添加剂或新型食品审批申请。

## 七、市场发展趋势与展望

### （一）发展趋势

1. **从"减糖"到"重建甜味"**：食品行业正从简单的减糖策略转向通过天然甜味蛋白等成分"重建甜味"体验。蜂蜜松露甜蛋白的纯净口感和高甜度正契合这一趋势【附件1】。

2 **GLP-1驱动的配方革命**：GLP-1类减肥药正在改变消费者的饮食模式和口味偏好。食品企业需要重新调整产品配方以吸引这一快速增长的人群。甜味蛋白凭借对血糖无影响、对肠道健康有益的特性，处于有利地位【附件5】。

3. **发酵技术的工业化成熟**：精密发酵技术正在快速成熟，FAAS（发酵即服务）等平台化商业模式降低了行业门槛，加速了新产品从研发到上市的周期【附件2】。

4. **清洁标签与透明度**：全球消费者对食品标签透明度的要求在提高。天然来源、简单配料、无人工添加剂的甜味蛋白产品具有显著的差异化优势【附件7】。

5. **亚太市场的增长潜力**：中国、日本、韩国等亚洲国家消费者对天然甜味剂和功能性食品的需求快速增长。蜂蜜松露作为高端食材在亚洲市场具有一定的认知基础，有利于市场开拓。

## 八、结论与建议

### （一）基本结论

1. **市场前景良好**：全球甜味蛋白市场处于快速增长阶段，预计2026至2033年复合增长率达10.3%，蜂蜜松露甜蛋白作为该领域的创新产品，具有广阔的发展空间。

2. **技术壁垒较高**：精密发酵技术路线具有较高的技术门槛，MycoTechnology在该领域已建立起一定的先发优势和专利壁垒。

3. **政策窗口期**：GLP-1减肥药带来的食品行业变革为天然甜味蛋白创造了历史性的市场窗口期。

4. **安全性有保障**：已完成的毒理学和安全性研究表明，蜂蜜松露甜蛋白具有良好的安全性特征，为其市场准入提供了科学依据。

### （二）相关建议

1. **关注监管动态**：密切跟踪美国FDA、欧盟EFSA、中国国家卫健委等监管机构对甜味蛋白的审批进展和标准制定。

2. **提前布局专利**：关注甜味领域的专利布局情况，防范知识产权风险。

3. **储备技术能力**：关注精密发酵技术在甜味蛋白生产中的应用进展，评估相关技术的可引进性和可转化性。

4. **把握市场时机**：鉴于GLP-1药物市场的快速扩张，建议相关企业关注甜味蛋白在下游食品饮料中的应用机会，适时布局产品研发和市场合作。

5. **加强风险防控**：针对产能、成本、监管等核心风险因素，制定相应的风险应对预案。
