# 第一部分：太平再保险

## 公司概况

| 项目 | 数据 |
|------|------|
| 公司全称 | 太平再保险有限公司 |
| 母公司 | 中国太平保险集团 |
| 总部 |香港 |
| 中国区 | 太平再保险（中国）有限公司（北京） |
| 2025年保费收入（估算） | 约60-80亿港元 |
| 网络覆盖 | 亚洲、欧洲、北美 |

## 战略定位

依托中国太平集团的跨境经营优势，定位为"立足港澳、辐射亚太"的区域型再保险龙头。

## 核心业务

1. **财产再保险**
   - 火险、水险、工程险
   - 意外险、责任险
   - 特殊风险

2. **寿险再保险**
   - 保障型寿险再保险
   - 健康险再保险
   - 年金再保险

3. **创新业务**
   - 大湾区跨境保险创新   - "一带一路"项目保险
   - 海南自贸港特殊风险

## 战略方向（推测）

- 以香港为基地，服务大湾区一体化进程中的跨境风险转移需求
- 依托中国太平集团在东南亚的业务网络，拓展区域再保险市场
- 关注海南自贸港建设中的保险创新机遇

# 第二部分：人保再保险

## 公司概况

| 项目 | 数据 |
|------|------|
| 公司全称 | 人保再保险股份有限公司 |
| 成立时间 | 201年 |
| 母公司 | 中国人保集团 |
| 2025年保费收入（估算） | 约40-60亿元 |

## 战略定位

依托人保集团资源，定位"集团内部风险分流+外部市场化拓展"双轮驱动。

## 核心业务

1. **财产险再保险**
   - 车险再保险
   - 企财险、责任险
 - 农业保险再保险

2. **人身险再保险**
   - 寿险再保险
   - 意外健康险再保险

3. **政策性业务**
   - 政策性农业保险再保险
   - 巨灾风险再保险

## 战略方向（推测）

- 深入承接人保财险、人保寿险的分保需求，发挥集团协同效应
- 逐步扩大非关联业务占比，提升独立市场化经营能力
- 在农业保险再保险和巨灾保险领域形成特色优势
- 关注科技保险、网络安全保险等新兴领域的再保险机会
