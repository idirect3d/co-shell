# 附件9 汉诺威再保险（Hannover Re）-中国区业务概况与战略方向

- **网站：** 汉诺威再保险集团官网（www.hannover-re.com）

- **采集日期：** 2026年月26日

## 公司概况

汉诺威再保险（Hannover Re）于1966年，总部位于德国汉诺威，是全球**第三大**再保险公司，在寿险再保险领域全球领先。

| 项目 | 数据|------|------|
| 集团总部 | 德国汉诺威 |
| 中国区总部 | 上海 |
| 集团2024年保费收入 | 约330亿欧元 |
| 全球排名 | 第三大 |
| 核心优势领域 | 寿险与健康险再保险（全球第一） |

## 中国区业务布局

1. **寿险再保险**
   - 保障型寿险再保险（定期寿险、重疾险）
   - 储蓄型寿险再保险
   - 年金与长寿风险

2. **健康险再保险**
   - 重大疾病保险再保险
   - 医疗保障再保险
   - 健康管理服务支持

3. **财产险再保险**
   - 财产险与意外险
   - 信用与保证保险
   - 特殊风险

4. **数字化解决方案**
   - eBUTION数字化核保系统
   - 风险数据分析和自动化承保工具

## 核心竞争优势

1. **寿险再保险全球领先**：在寿险和健康险再保险领域全球市占率第一
2. **数字化承保（eBUTION）**：全球领先的数字化的自动核保平台，已在多家直保公司部署
3. **IFRS17专业知识**：在IFRS17实施方面积累深厚经验
4. **高效经营**：以运营效率和盈利能力著称

## 未来1-3年战略方向

| 战略方向 | 具体措施 |
|---------|---------|
| 寿险再保险深耕 | 巩固保障型寿险再保险优势，聚焦重疾险和医疗险 |
| eBUTION本土化 | 推动数字化核保系统eBUTION在中国的深度应用和定制化 |
| IFRS17机遇 | 利用全球经验协助客户应对新会计准则，优化再保险安排 |
| 健康险创新 | 发展高端医疗险、健康管理服务再保险方案 |
| 长寿风险管理 | 应对人口老龄化趋势下的长寿风险转移需求 |
| 绿色保险 | 探索ESG相关的再保险解决方案 |
