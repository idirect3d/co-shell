## UC-0001: StdioIO.ReadKey() 在 raw terminal 下正确阻塞等待按键

### 测试目标
验证 StdioIO.ReadKey() 在启用 raw console mode 后能正确阻塞等待用户按键输入。

### 前置条件
- 操作系统: 任何支持 raw terminal 的平台
- 编译: `go build ./...` 通过

### 测试步骤
1. 创建测试程序，调用 `NewStdioIO()` 获取 StdioIO 实例
2. 调用 `ReadKey()` 方法
3. 观察程序行为

### 预期结果
- ReadKey() 调用 MakeRaw() 启用 raw terminal mode 后再调用 os.Stdin.Read()
- 调用后阻塞等待用户按键
- 用户按键后返回正确的 byte 值

### 验证方法
- 代码审查确认实现正确
- `go vet ./...` 通过
